# templates

Source templates used by `install.sh` to generate AI tool integrations. Each subdirectory targets one AI tool.

| Path | Description |
|---|---|
| [`claude/`](claude/) | Claude Code command template |
| [`codex/`](codex/) | Codex agent skill template |
| [`copilot-cli/`](copilot-cli/) | GitHub Copilot CLI skill template |
| [`copilot-ide/`](copilot-ide/) | GitHub Copilot IDE prompt template |
| [`extra-catalog/`](extra-catalog/) | Starter scaffold for building a custom extra-catalog |
